## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries, realizado por Vinicius. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos especí­ficos. A seguir, serão descritas as etapas do projeto:

Etapa 1:
- Amazon S3
- Armazenamento de dados e documentos em nuvem.
- Substituição de servidores de arquivos físicos e HDs externos para guardar receitas, laudos e documentos regulatórios, pagando apenas pelo espaço utilizado e eliminando custos de manutenção de hardware.

Etapa 2:
- Amazon EC2
- Servidores virtuais escaláveis.
- Migração do sistema de gestão (ERP) da empresa para a nuvem. Isso permite desligar servidores antigos que consomem muita luz e espaço, ajustando a potência do servidor conforme a necessidade da operação.

Etapa 3:
- Amazon RDS
- Gerenciamento de bancos de dados relacionais.
- Automatização da administração dos dados de estoque e clientes. Reduz o custo operacional ao automatizar backups e atualizações, evitando que um funcionário precise gastar horas do dia cuidando da manutenção do banco.
